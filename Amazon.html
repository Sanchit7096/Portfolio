Amazon Clone

Year: 2025 HTML, CSS, JavaScript

Description

Amazon Frontend Clone is a project made using HTML, CSS, and JavaScript that looks and works like the Amazon website. It has important features like a search bar, product categories, product cards, add-to-cart option, and a responsive design that works on all devices. This project helps learners understand how big e-commerce websites are designed and how their frontend works. By making this clone, you can practice layouts, styling, and adding interactive features with JavaScript in a fun and practical way.

Key Features

  • Navigation Bar
  • Product Categories
  • Product Cards
  • Responsive Design

My Role

  • Designed the frontend layout using HTML and CSS to match Amazon’s look.
  • Developed responsive pages that work on desktop and mobile devices.
  • Created product cards and categories with proper styling and structure.
