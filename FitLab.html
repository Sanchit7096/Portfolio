FitLab

Year: 2025 React

Description

FitLab is a simple React.js gym website created for practice. It shows a clean and attractive design with gym interior images, exercise sections, and fitness content to give users a real gym-like feel. The project helped me practice React components, props, and responsive design while building a modern website layout.

Key Features

  • Modern React UI – Built fully with React.js components.
  • Responsive Design – Works on mobile, tablet, and desktop
  • Hero Section – Eye-catching gym banner with intro text.
  • Image Gallery – Gym interiors and fitness visuals to make the site attractive.
  • Navigation Bar – Simple header with logo and menu.
  • Footer Section – Contact details and quick links.

My Role

  • Designed and built the frontend of the website using React.js.
  • Used components and props to create reusable sections.
  • Made the design responsive so it works on all devices.
  • Focused on layout and styling for a modern and clean look
